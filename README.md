# 句存 - 网页轻笔记

一个优雅高效的 Chrome 扩展，用于快速收藏、分类整理网页内容，支持搜索与多格式导出。

![License](https://img.shields.io/badge/license-MIT-green) ![Version](https://img.shields.io/badge/version-1.1.0-blue)

## 📋 功能特性

- **快速收藏** - 在任何网页上右键选中文字，一键保存为笔记
- **本地存储** - 所有笔记存储在本地，隐私安全
- **实时编辑** - 点击笔记内容即可编辑，自动保存修改
- **分类整理** - 新建、重命名、改色或删除分类，并可快速移动单条笔记
- **搜索筛选** - 按内容、来源或分类搜索，快速查看某一分类和最近添加的笔记
- **智能导出** - 支持导出为 TXT、Word、PDF 三种格式
- **灵活选项** - 导出时可选择是否包含时间戳和来源链接
- **设计简洁** - 现代 UI设计

## 📖 使用说明

### 管理笔记

| 操作 | 方式 |
|------|------|
| **添加笔记** | 网页右键菜单 → 选中文字 → 添加笔记 |
| **新建笔记** | 管理页右上角 → 新建笔记 |
| **编辑笔记** | 点击笔记内容 → 直接输入修改 → 点击外部自动保存 |
| **移动分类** | 使用笔记卡片底部的分类菜单 |
| **管理分类** | 左侧“管理分类” → 修改名称、颜色或删除分类 |
| **删除笔记** | 鼠标悬停卡片 → 点击右上角 × 按钮 |
| **清空全部** | 点击右上角 "清空" 按钮（需确认） |

### 导出功能

1. **点击导出按钮** - 打开导出菜单
2. **选择格式**
   - **TXT**
   - **Word**
   - **PDF**

3. **配置选项**
   - ☑ **包含分类** - 每条笔记附带分类名称
   - ☑ **包含时间** - 每条笔记附带时间戳
   - ☑ **包含来源** - 每条笔记附带源页面链接
