# 句存 - 网页轻笔记

一个优雅高效的 Chrome 扩展，用于快速收藏、分类整理网页内容，支持搜索与多格式导出。

![License](https://img.shields.io/badge/license-MIT-green) ![Version](https://img.shields.io/badge/version-1.2.0-blue)

## 📋 功能特性

- **快速收藏** - 在任何网页上右键选中文字，一键保存为笔记
- **本地存储** - 所有笔记存储在本地，隐私安全
- **实时编辑** - 点击笔记内容即可编辑，自动保存修改
- **分类整理** - 新建、重命名、改色或删除分类，并可快速移动单条笔记
- **搜索筛选** - 按内容、来源或分类搜索，快速查看某一分类和最近添加的笔记
- **智能导出** - 支持导出为 TXT、Word、PDF 三种格式，PDF 在扩展内直接生成
- **实时预览** - 导出前即可查看最终分页、排版和内容效果
- **自定义排版** - 支持纸张、方向、页边距、模板、字体、字号、行距和页码设置
- **设计简洁** - 现代 UI设计

## 📖 使用说明

### 管理笔记

| 操作 | 方式 |
|------|------|
| **添加笔记** | 网页右键菜单 → 选中文字 → 添加笔记 |
| **新建笔记** | 管理页右上角 → 新建笔记 |
| **编辑笔记** | 点击笔记内容 → 直接输入修改 → 点击外部自动保存 |
| **移动分类** | 使用笔记卡片底部的分类菜单 |
| **管理分类** | 左侧“管理分类” → 修改名称、颜色或删除分类 |
| **删除笔记** | 鼠标悬停卡片 → 点击右上角 × 按钮 |
| **批量操作** | 点击右上角“批量选择”，勾选笔记后可导出或删除所选内容 |

### 导出功能

1. **点击导出按钮** - 打开导出工作台
2. **选择格式**
   - **TXT**
   - **Word**
   - **PDF**

3. **配置 PDF 排版**
   - 选择 A4、Letter 或 A5，以及纵向或横向页面
   - 选择简约、文稿或卡片模板
   - 调整字体、字号、行距和页边距；可手动输入或授权读取本机已安装字体
   - 选择是否显示分类、时间、来源和页码，PDF 与 Word 会保留可点击的来源链接
4. **实时预览并导出** - PDF 文件仅在本地生成，不再依赖浏览器打印界面
